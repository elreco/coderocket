Learn how to purchase, download, and customize components from the CodeRocket Marketplace.

## Getting Started

### Account Requirements
- Free or premium CodeRocket account
- Valid payment method (credit/debit card)
- Email for purchase receipts

### Browsing the Marketplace

<Steps>
  <Step title="Visit the Marketplace">
    Navigate to the [Marketplace](https://coderocket.app/marketplace) from the sidebar
  </Step>
  <Step title="Filter & Search">
    Use categories and search to find components that match your needs
  </Step>
  <Step title="Preview Components">
    Click on components to see detailed previews, descriptions, and demos
  </Step>
</Steps>

## Making a Purchase

### 1. Select Your Component
- Browse by category or search for specific needs
- Read the description and view preview
- Check the demo URL if available
- Review pricing and seller information

### 2. Purchase Process
<Steps>
  <Step title="Click Purchase">
    Click the "Purchase for $X.XX" button on the component detail page
  </Step>
  <Step title="Secure Checkout">
    You'll be redirected to Stripe for secure payment processing
  </Step>
  <Step title="Complete Payment">
    Enter your payment details and complete the transaction
  </Step>
  <Step title="Access Component">
    You'll be redirected back with immediate access to your purchase
  </Step>
</Steps>

### 3. What You Get
When you purchase a component, you receive:
- **Full source code** access
- **All assets** (images, icons, etc.)
- **Component files** in your preferred framework
- **Documentation** and usage instructions
- **Lifetime access** to your purchase

## Accessing Your Purchases

### My Purchases Page
View all your purchased components:
- Go to **My Purchases** in the sidebar
- See purchase history and dates
- Access download and export options

### Download Options
<CardGroup cols={2}>
  <Card title="Direct Download" icon="download">
    Download component files as a ZIP archive for local development
  </Card>
  <Card title="GitHub Export" icon="github">
    Export directly to a GitHub repository for version control
  </Card>
</CardGroup>

## Using Your Purchases

### What's Included in Downloads

```
component-name/
├── src/
│   ├── components/     # React/Vue/HTML files
│   ├── styles/        # CSS/Tailwind styles
│   └── utils/         # Helper functions
├── assets/
│   ├── images/        # Component images
│   └── icons/         # Icon files
├── package.json       # Dependencies
├── README.md         # Usage instructions
└── preview.html      # Live preview
```

### Installation & Setup

<Steps>
  <Step title="Download Component">
    Download the ZIP file from your purchases page
  </Step>
  <Step title="Extract Files">
    Extract to your project directory or desired location
  </Step>
  <Step title="Install Dependencies">
    Run `npm install` to install required packages
  </Step>
  <Step title="Import Component">
    Follow the README instructions to integrate into your project
  </Step>
</Steps>

## GitHub Integration

### Exporting to GitHub

<Steps>
  <Step title="Connect GitHub">
    Link your GitHub account in account settings if not already connected
  </Step>
  <Step title="Choose Repository">
    Select existing repository or create a new one
  </Step>
  <Step title="Auto-Export">
    Component files are automatically committed to a new branch
  </Step>
  <Step title="Clone & Develop">
    Clone the repository and start customizing
  </Step>
</Steps>

### Benefits of GitHub Export
- **Version Control**: Track all your changes
- **Collaboration**: Work with team members
- **Backup**: Your components are safely stored
- **CI/CD**: Integrate with deployment pipelines

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Can I modify purchased components?">
    Yes! You get full source code access and can modify components however you need for your projects.
  </Accordion>

  <Accordion title="Can I use components commercially?">
    Yes, purchased components can be used in commercial projects without additional licensing fees.
  </Accordion>

  <Accordion title="What if I lose my files?">
    You have lifetime access to your purchases. You can re-download anytime from your My Purchases page.
  </Accordion>

  <Accordion title="Do I get updates to components?">
    You get the component version available at time of purchase. Updates may be released as new versions.
  </Accordion>

  <Accordion title="Can I resell components?">
    No, you cannot resell purchased components. They're licensed for your use only.
  </Accordion>
</AccordionGroup>
