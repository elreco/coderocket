The CodeRocket Marketplace is a platform where premium users can sell their AI-generated components and buyers can purchase high-quality, customizable components for their projects.

## What is the Marketplace?

The Marketplace is a community-driven platform that allows:

- **Sellers**: Premium users to monetize their best components
- **Buyers**: Anyone to purchase and customize professional components
- **Creators**: Access to a wide variety of ready-made solutions

## How It Works

### For Sellers
1. Create high-quality components using CodeRocket
2. Set them as private
3. List them in the marketplace with pricing
4. Earn 70% of each sale (30% platform commission)

### For Buyers
1. Browse the marketplace by category
2. Preview components before purchasing
3. Buy with secure Stripe payments
4. Download or export to GitHub
5. Customize the source code

## Categories

The marketplace includes these component categories:

- **Landing Pages**: Complete website landing pages
- **UI Components**: Reusable interface elements
- **Email Templates**: Professional email designs
- **Web Templates**: Full website templates
- **Mobile Templates**: Mobile-optimized layouts
- **Dashboards**: Admin and analytics interfaces
- **E-commerce**: Shopping and product pages
- **Portfolios**: Personal and professional portfolios
- **Forms**: Contact, signup, and data collection forms
- **Navigation**: Headers, sidebars, and menu systems

## Requirements

### To Sell
- Active CodeRocket premium subscription
- Private components only (not publicly listed)
- Original, high-quality work

### To Buy
- CodeRocket account (free or premium)
- Valid payment method

## Commission Structure

<Info>
  CodeRocket takes a 30% commission on all marketplace sales. Sellers receive 70% of the sale price.
</Info>

This commission helps us:
- Maintain the platform and marketplace
- Process payments securely
- Provide customer support
- Continue developing new features
