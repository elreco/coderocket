The **CodeRocket Marketplace** is a platform where you can buy and sell high-quality AI components. Simple, secure, and profitable.

## 🛒 Buying components

<Steps>
  <Step title="Explore">
    Browse components by category or use the search function
  </Step>
  <Step title="Preview">
    Test the component before purchase with the interactive demo
  </Step>
  <Step title="Purchase">
    Secure payment with Stripe. Instant download.
  </Step>
  <Step title="Use">
    Get the complete source code + GitHub export option
  </Step>
</Steps>

<Note>
  **No subscription required** to buy components!
</Note>

## 💰 Selling your creations

<Steps>
  <Step title="Create">
    Build quality components with CodeRocket
  </Step>
  <Step title="List">
    Put your component for sale on the marketplace
  </Step>
  <Step title="Earn">
    **70% of each sale** directly to your account
  </Step>
</Steps>

<Warning>
  **Requirement to sell:** Active CodeRocket Premium subscription
</Warning>
