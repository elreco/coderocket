## Key Features

### 🔄 Automatic Synchronization
Every time you generate a new version of your project with AI, CodeRocket automatically:
- Creates a new commit in your GitHub repository
- Updates all project files
- Generates appropriate commit messages
- Maintains version history

### 📤 Manual Sync Options
You have full control over when and how to sync:

<Tabs>
  <Tab title="Pull from GitHub">
    Download the latest changes from your GitHub repository and create a new version in CodeRocket.

    **Use when**: You've made changes directly in GitHub and want to bring them into CodeRocket.
  </Tab>

  <Tab title="Push to GitHub">
    Upload your current CodeRocket version to GitHub only if the web version is newer.

    **Use when**: You want to push changes while respecting GitHub modifications.
  </Tab>

  <Tab title="Force Push">
    Upload your current CodeRocket version to GitHub, overriding any GitHub changes.

    **Use when**: You want to ensure your CodeRocket version becomes the source of truth.
  </Tab>
</Tabs>

### 🏗️ Repository Structure
CodeRocket automatically creates a professional repository structure:

```
your-repository/
├── README.md              # Project documentation
├── package.json           # Dependencies and scripts
├── src/
│   ├── App.js            # Main application file
│   ├── components/       # React components
│   └── styles/           # CSS and styling files
└── public/
    └── index.html        # HTML template
```

### 🔐 Security & Authentication
- Secure OAuth integration with GitHub
- Repositories are created under your GitHub account
- You maintain full ownership and control
- Access tokens are encrypted and stored securely

  <Accordion title="Team Collaboration">
    1. Create and sync your project to GitHub

  </Accordion>
</AccordionGroup>
