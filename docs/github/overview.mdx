## What is GitHub Integration?

The GitHub integration allows you to sync your AI-generated components with GitHub repositories. This gives you version control and lets you continue developing your projects in your favorite code editor.

## How It Works

1. **Generate** a component with AI in CodeRocket
2. **Connect** your GitHub account
3. **Create** a repository for your component
4. **Sync** automatically - every AI change creates a new commit
5. **Continue** developing in GitHub or pull changes back to CodeRocket

<Note>
**Important**: To modify your code, you need to edit files directly in GitHub or in your local development environment. CodeRocket cannot edit code that has been modified outside the platform.
</Note>

## Key Features

### 🔄 Two-Way Sync
- **Push to GitHub**: Send your CodeRocket changes to GitHub
- **Pull from GitHub**: Bring external changes back into CodeRocket

### 📁 Clean Repository Structure
Your repository will include:
- Your component code (React, HTML, etc.)
- A README with instructions
- Package.json with dependencies
- Professional project structure

### 🛡️ Safe & Secure
- Uses GitHub OAuth (no passwords stored)
- Repositories are created under your GitHub account
- You maintain full ownership and control

## Getting Started

Ready to sync your components with GitHub?

<Steps>
  <Step title="Connect GitHub">
    Go to your account settings and connect your GitHub account
  </Step>
  <Step title="Create Repository">
    Open any component and click the settings gear to set up GitHub sync
  </Step>
  <Step title="Start Syncing">
    Enable automatic sync and continue iterating with AI
  </Step>
</Steps>
