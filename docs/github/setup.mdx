## What You Need

<Check>A CodeRocket account</Check>
<Check>A GitHub account</Check>
<Check>At least one component created in CodeRocket</Check>

## Step 1: Connect Your GitHub Account

<Steps>
  <Step title="Go to Account Settings">
    Click your profile picture → **Account Settings**
  </Step>

  <Step title="Connect GitHub">
    Find the GitHub section and click **Connect GitHub Account**

    You'll be redirected to GitHub to authorize CodeRocket
  </Step>

  <Step title="Grant Permissions">
    Allow CodeRocket to:
    - Create repositories
    - Read and write repository contents
    - Access basic profile information
  </Step>

  <Step title="Return to CodeRocket">
    You'll be redirected back and see your GitHub username displayed
  </Step>
</Steps>

## Step 2: Set Up a Component Repository

<Steps>
  <Step title="Open Component Settings">
    Go to any component and click the **Settings** button (⚙️)
  </Step>

  <Step title="Create Repository">
    In the GitHub Sync section, click **Create GitHub Repository**

    CodeRocket will create a repository with your component's code
  </Step>

  <Step title="Enable Automatic Sync">
    Toggle **Enable GitHub Sync** to automatically sync future AI iterations
  </Step>
</Steps>

## What Gets Created

Your new GitHub repository will contain:
- **Your component code** (React/HTML files)
- **README.md** with project description
- **package.json** with dependencies
- **Professional folder structure**

<Info>
Repository names follow the pattern: `coderocket-{your-component-name}`
</Info>

## Verify Everything Works

1. **Check GitHub**: Visit your GitHub profile to see the new repository
2. **Make a change**: Ask the AI to modify your component
3. **Check commits**: See the new commit appear in GitHub

## Troubleshooting

<AccordionGroup>
  <Accordion title="Connection Failed">
    - Make sure you're logged into GitHub
    - Try disconnecting and reconnecting
    - Check your internet connection
  </Accordion>

  <Accordion title="Repository Creation Failed">
    - The repository name might already exist
    - Try with a different component
    - Wait a few minutes and try again
  </Accordion>

  <Accordion title="Still Having Issues?">
    Join our [Discord](https://discord.gg/t7dQgcYJ5t) and ask for help in the **#support** channel. Our team is ready to help!
  </Accordion>
</AccordionGroup>
