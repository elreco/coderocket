## Setup your account

Welcome to CodeRocket! Let's get you up and running with your first AI-generated web application.

<AccordionGroup>
  <Accordion title="Step 1: Create your account">
    Visit [CodeRocket](https://coderocket.app) and sign up for a free account. You can use:
    - Email and password
    - Google authentication
    - GitHub authentication (recommended for GitHub integration)
  </Accordion>

  <Accordion title="Step 2: Choose your subscription">
    CodeRocket offers different tiers:
    - **Free**: Limited generations per month
    - **Pro**: Unlimited generations + GitHub integration
    - **Enterprise**: Team features + priority support
  </Accordion>
</AccordionGroup>

## Create your first project

<Steps>
  <Step title="Start a new component">
    Click the "New Component" button on your dashboard and describe what you want to build.

    Example prompts:
    ```
    Create a modern landing page for a SaaS product
    Build a todo app with React
    Make a pricing table with 3 tiers
    ```
  </Step>

  <Step title="Choose your framework">
    Select the framework that best fits your needs:
    - **React** - For modern component-based apps
    - **Vue.js** - For progressive JavaScript applications
    - **Angular** - For enterprise TypeScript applications
    - **Svelte** - For high-performance reactive apps
    - **HTML/CSS** - For simple websites
  </Step>

  <Step title="Generate and iterate">
    Watch as CodeRocket generates your application in real-time. You can then:
    - Ask for modifications: "Make the button blue"
    - Add features: "Add a contact form"
    - Change styling: "Use a dark theme"
  </Step>

  <Step title="Preview and export">
    Use the live preview to test your application, then export your code or sync to GitHub.
  </Step>
</Steps>

## Tips for better results

<Tip>
  **Be specific**: The more detailed your prompt, the better the AI can understand your requirements.

  Instead of: "Make a website"
  Try: "Create a modern portfolio website with a hero section, project gallery, and contact form"
</Tip>

<Warning>
  Remember to save your work! While CodeRocket auto-saves your progress, we recommend setting up GitHub integration for proper version control.
</Warning>
